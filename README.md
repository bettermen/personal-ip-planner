# personal-ip-planner

个人IP规划全流程助手 — WorkBuddy Skill

用户输入IP名称与定位方向，自动从 7 大维度进行可行性分析，生成含雷达图、竞品矩阵和12个月行动路线图的交互式 HTML 报告。

## 7 维分析框架

| 维度 | 权重 | 分析要点 |
|------|------|----------|
| 🏢 竞品格局 | 15% | 同类IP数量、头部账号、差异化机会 |
| 👥 目标用户 | 20% | 用户画像、痛点需求、付费意愿 |
| 🎯 品牌定位 | 15% | 名称评估、定位清晰度、记忆点 |
| 📝 内容策略 | 20% | 内容矩阵、爆款选题、发布节奏 |
| 📱 平台策略 | 10% | 主平台选择、多平台矩阵协同 |
| 💰 变现路径 | 15% | 短中长期变现模式设计 |
| ⚠️ 风险控制 | 5% | 潜在风险、应对策略 |

## 触发方式

```
个人IP规划：做一个AI产品经理的IP，叫「AI产品说」
帮我评估做'钓鱼老王'这个IP可行吗
分析'职场妈妈Lisa'这个个人品牌的可行性
```

## 安装

```bash
# 复制到 WorkBuddy skills 目录
cp -r personal-ip-planner ~/.workbuddy/skills/
```
